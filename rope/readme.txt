Attribution
-----------
This dataset is part of a submission to the NeurIPS 2021 Datasets and Benchmarks Track.
It is a subset of the full MVTec 3D-AD dataset and intended for review purposes only.
Our intention was to limit the file size for the reviewers to download. The complete
dataset will be made availble for reviewers upon request and to the public upon
acceptance of the corresponding submission.
